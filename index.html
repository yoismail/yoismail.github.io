End-to-End Cloud Data Pipeline for E-commerce Analytics

Building a scalable batch and streaming analytics platform on the cloud: I designed and implemented a full end-to-end data engineering pipeline to ingest, process, and analyze e-commerce transaction data. Data was sourced from multiple inputs including CSV files, APIs, and real-time event streams. Batch processing was handled using Apache Spark, while streaming data was processed with Apache Kafka.
